HTML Tags

Tag Name Description
<li> List items Defines a type of list(ordered/unordered)
<p> Paragraph Defines a paragraph
<h1> Title header Defines a title of a page
<em> Emphasize Emphasizes text
<figcaption> Figure caption Caption to a figure
<embed> Embed Picks any media type
<label> Label Identifies an input type
<input> Input Takes inputs depending on types
<form> Form Allows users to input and send info
<textarea> Textarea Section to input info

CSS Tags

Tag Description
!important Overrides any style
* Selects and applies styling to all elements in stylesheet
rgba() Red, Green, Blue, apha color property
hsla() Hue, Saturation, Lightness, alpha color property
font-weight Weight of font
text-align Aligns text horizontally
padding Distance between text and border
margin Distance from outside the border
overflow Controls content that spill or overflow outside the container
relative Moves elements relative to its original position
offset properties top, bottom, right, left
absolute Positioned relative to its closest parent container
sticky Relative but when on specified offset, it sticks to that position
positioning properties static, relative, fixed, sticky
z-index Specifies how far back or far forward an element should appear when they overlap
flex Short form to declare flex-grow, flex-shrink and flex-basis
flex-direction Determines direcetion of flexbox (ie. row, row-reverse, column, column-reverse)
flex-flow Used to declare both flex-wrap & flex-direction
grid-template-columns Defines number of columns and their sizes
grid-template-rows Defines number of rows and their sizes
grid-template Defines number and size for rows and columns
grid-template-areas Specifies the position of semantic properties on a grid
minmax() Defines the size of a grid at default
grid-gap Applies gaps in between rows and columns
grid-area Sets starting and ending positions for rows and columns (ie. grid-row-start, grid-column-start, grid-row-end, grid-column-end)
align-content property Used to position contents on the cross-axis (eg. flex-start, flex-end, center, space-between, space-around, stretch)
justify-content Used to position contents/groups on the main-axis
justify-items Used to position/spread individual items on the main-axis
justify-self Specifies how a single element should be positioned on the main-axis
